# svelte-training

実践Svelte入門(Svelte編)

## premise

* nodeのインストール

  ```bat
  # Versions check
  $ volta --version
  $ node -v
  $ npm -v
  ```

* githubのページ作成

## install and settings

* git clone
* `settings.code-workspace`の生成
* Svelteのセットアップ

  ```bat
  # Create from vite
  $ npm create vite@latest . -- --template svelte-ts

  # install
  $ npm install
  ```

* 稼働確認

  ```bat
  # run dev
  $ npm run dev
  ```

## usege
